# DKPAuctionBidder
Small UI to bid in DKP Auctions created by SotA.

Automatically pops up, when an auction is started by the SOTA Master.
___

#### How to Install

1. Click "Clone or Download" and download the ZIP file
2. Extract the ZIP file
3. Rename the folder "DKPAuctionBidder-master" to "DKPAuctionBidder"
4. Move it to your WoW Installation/Interface/AddOns
5. ????
6. Profit!!!!
